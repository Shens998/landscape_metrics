# Changelog

All notable changes to this project are documented here. The project remains in alpha; interfaces and metric contracts may evolve with documented regression tests.

## Unreleased

### Added

- Exact memory-bounded tiled GeoTIFF calculations for the frozen v0.1 metric set.
- Public metric-column selection, bilingual methods documentation, and reproducible chunked benchmark.
- Standard metric names, abbreviations, Chinese names, and LaTeX formulas in metric cards.

### Changed

- Release engineering now validates tests, static checks, type checks, and package construction.
