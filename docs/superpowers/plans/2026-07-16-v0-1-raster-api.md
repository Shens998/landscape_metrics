# Open Landscape Metrics v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 发布一个独立实现、面向单波段 LULC 分类 GeoTIFF 与二维整数数组的 Python API，可精确计算冻结的基础斑块、类别和景观指标，并保证内存与分块路径数值一致。

**Architecture:** 以严格输入验证、可复用拓扑基础量和纯指标函数为核心。小栅格走内存路径；大型 GeoTIFF 使用 Rasterio 窗口、磁盘映射标签、并查集和临时 SQLite 聚合，在不依赖专有计算引擎的前提下得到同一结果。对外只暴露 `Landscape` 与 DataFrame 结果；所有规则、来源和运行参数写入元数据。

**Tech Stack:** Python 3.11+, NumPy, SciPy (`ndimage.label`), pandas, Rasterio/GDAL, PyYAML, pytest, Hypothesis, Ruff, mypy, MkDocs Material, GitHub Actions.

## Global Constraints

- 仅支持单波段、整数分类 GeoTIFF 和二维整数 NumPy 数组；不实现 CLI、GUI、矢量、移动窗口、FRAGSTATS 工程读取、Core Area、IIC 或 PC。
- 实现、文档、测试与示例独立撰写；不调用 FRAGSTATS，不复制其源代码、工程格式、界面或文档文字。
- GeoTIFF 必须有投影 CRS、北向上且无旋转/剪切；像元宽高可不同，面积和共享边长须分别使用 `abs(transform.a)` 与 `abs(transform.e)`。
- `NoData` 是排除区，不视为背景或内部边；背景必须是显式类别值。默认将景观外边界计为边。
- `connectivity` 只决定斑块连通，取值为 4 或 8；所有周长与类别邻接只计算共享边，绝不把角接触当边。
- 默认严格模式：拒绝浮点分类值、二维以外数组、地理 CRS、未知像元尺度、无效连通和临时空间不足；不得静默重投影、重采样、重编码或近似。
- 面积单位为输入 CRS 的平方单位；长度为 CRS 单位；`patch_density` 的统一单位为每 100 ha，`edge_density` 为 m/ha，前提是 CRS 单位为米。非米投影坐标须显式传入 `linear_unit_to_m`，否则报错。
- 结果必须含 `metadata`：输入摘要、类别、NoData、CRS、像元宽高、连通/边界规则、执行路径、块大小、临时文件版本、包版本与指标版本。
- 所有计算变更以测试先行；每个任务独立通过测试并独立提交。任何数值规则变更须增添回归测试并写入 CHANGELOG。

## Frozen v0.1 metric contract

在 Task 2 完成后，以下标识、单位和公式不再改名或改变含义。设斑块 (p) 的面积为 (a_p)、周长为 (P_p)、类别 (i) 总面积为 (A_i)、景观有效面积为 (A)、类别数为 (m)，类别面积比例为 (q_i=A_i/A)，同类共享边数为 (g_{ii})，类别 (i) 可达到的最大同类共享边数为 (max g_{ii})。

| 层级 | 标识 | 定义 / 单位 |
|---|---|---|
| patch | `area` | (a_p)，CRS 平方面积单位 |
| patch | `perimeter` | (P_p)，只计有效类别间共享边及景观外边界，CRS 长度单位 |
| patch | `shape_index` | (P_p/(2\sqrt{\pi a_p}))，无量纲 |
| patch | `perimeter_area_ratio` | (P_p/a_p)，1/长度 |
| patch | `fractal_dimension` | (2\ln(0.25P_p)/\ln(a_p))；当 (a_p\le1) 个等面积像元时为 `NaN) |
| patch | `radius_of_gyration` | 所有该斑块像元中心到质心距离的 RMS，长度单位 |
| class | `total_area` | (A_i) |
| class | `proportion_of_landscape` | (100q_i)，% |
| class | `number_of_patches` | 该类别斑块数 |
| class | `patch_density` | (N_i/A_{ha}\times100)，每 100 ha |
| class | `largest_patch_index` | (100\max(a_p)/A)，% |
| class | `total_edge` | 类别边长总和，长度单位 |
| class | `edge_density` | (E_i/A_{ha})，m/ha |
| class | `area_mean`, `area_sd`, `area_cv` | 斑块面积的样本均值、样本标准差（(n<2) 为 `NaN`）与 (100sd/mean) |
| class | `shape_index_mean` | 类别中斑块 `shape_index` 算术均值 |
| class | `aggregation_index` | (100g_{ii}/\max g_{ii})，%；(g_{ii}) 是同类共享边的**数量**，不是长度 |
| landscape | `total_area`, `number_of_patches`, `patch_density`, `edge_density`, `largest_patch_index` | 将所有类别按上表景观分母 (A) 汇总 |
| landscape | `shannon_diversity` | (-\sum_i q_i\ln q_i) |
| landscape | `shannon_evenness` | (H/\ln(m))；(m=1) 时为 0 |
| landscape | `simpson_diversity` | (1-\sum_iq_i^2) |
| landscape | `aggregation_index` | (100\sum_i g_{ii}/\sum_i\max g_{ii})，%；(g_{ii}) 是同类共享边的数量 |

指标卡必须至少引用：Shannon (1948), DOI 10.1002/j.1538-7305.1948.tb01338.x；Simpson (1949), DOI 10.1038/163688a0；Patton (1975), DOI 10.2307/3781151；He, DeZonia & Mladenoff (2000), DOI 10.1023/A:1008102521322。其余栅格斑块定义与半径/分形公式标为 `definition_reference`，使用 McGarigal & Marks (1995) USDA GTR PNW-GTR-351 的稳定链接与独立实现交叉参考 Hesselbarth et al. (2019), DOI 10.1111/ecog.04617；不得把这些定义性参考误标为每个公式的原始论文。

from dataclasses import dataclass
from pathlib import Path
from typing import Literal
from .errors import ConfigurationError, SpatialMetadataError

@dataclass(frozen=True, slots=True)
class GridSpec:
    width: int; height: int; pixel_width: float; pixel_height: float
    crs: str | None; linear_unit_to_m: float | None = None
    def __post_init__(self) -> None:
        if self.width < 1 or self.height < 1:
            raise SpatialMetadataError("width and height must be positive")
        if self.pixel_width <= 0:
            raise SpatialMetadataError("pixel_width must be positive")
        if self.pixel_height <= 0:

```python
def read_array(values, *, pixel_width, pixel_height, crs, nodata):
    if values.ndim != 2 or not np.issubdtype(values.dtype, np.integer):
        raise InvalidRasterError("input must be a two-dimensional integer categorical array")
    parsed = CRS.from_user_input(crs)
    if parsed.is_geographic:
        raise SpatialMetadataError("a projected CRS or explicit meter scale is required")
    return values.copy(), GridSpec(values.shape[1], values.shape[0], pixel_width, pixel_height, parsed.to_string())
```

For GeoTIFF, require `dataset.count == 1`, `dataset.crs is not None`, `dataset.transform.b == dataset.transform.d == 0`, `dataset.transform.a > 0`, and `dataset.transform.e < 0`. Return `abs(a)` and `abs(e)`; do not load a full raster in `inspect_geotiff`.
